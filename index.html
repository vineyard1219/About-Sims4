About Sims4

심즈4! 인생을 플레이하세요!

개요
장르시뮬레이션
출시일 : 2014년 9월 2일
이용 등급 : 15세이용가
퍼블리셔 : Electronic Arts
개발사 : The Sims Studio
지원 언어 : Český, Dansk, Deutsch (DE), English (US), Español (ES), Suomi, Français (FR), Italiano, 日本語, 한국어, Nederlands, Norsk, Polski, Português (BR), Русский язык, Svenska, 中文（繁體）
게임 링크 : 공식 사이트 포럼

시스템 요구 사항
최소 사양:
운영체제: Windows XP (SP3), Windows Vista (SP2), Windows 7 (SP1), Windows 8, Windows 8.1, Windows 10
프로세서: 1.8 GHz Intel Core 2 Duo, AMD Athlon 64 Dual-Core 4000+ 이상 (내장형 그래픽 칩셋을 사용하는 컴퓨터는 2.0 GHz Intel Core 2 Duo, 2.0 GHz AMD Turion 64 X2 TL-62 이상 필요)
RAM: 최소 2 GB RAM
하드 드라이브: 최소 14 GB 이상 여유 공간에 개인 콘텐츠 및 게임 저장용으로 1 GB 이상의 추가 공간
비디오: 128 MB 비디오 RAM, Pixel Shader 3.0 지원 지원하는 비디오 카드: NVIDIA GeForce 6600 이상, ATI Radeon X1300 이상, Intel GMA X4500 이상
DIRECTX: DirectX 9.0c 호환
사운드 카드: DirectX 9.0c 호환
입력: 키보드 및 마우스
온라인 연결: 제품을 활성화하려면 인터넷 연결 필수

권장 사양:
운영체제: 64 Bit Windows 7, 8, 8.1, 10
프로세서: Intel core i5 이상, AMD Athlon X4
비디오 카드: NVIDIA GTX 650 이상
메모리: 4 GB RAM
하드 드라이브: 18 GB 하드 드라이브 공간
